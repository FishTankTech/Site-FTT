FishTankTech is a software engineering and systems development practice built on the belief that good code should be maintainable, purposeful, and built to last — not patched together until the next sprint.

The work spans the full stack: web platforms that scale gracefully, data pipelines that run reliably, and analytical tooling that surfaces the signal in the noise. Every system is engineered to be understood, not just operated.

Outside the terminal, the practice extends into embedded systems and motorsports engineering under Activism Motorsports — where software meets asphalt and the feedback loop is measured in seconds.
